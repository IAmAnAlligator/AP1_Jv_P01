# AP1_Jv_P01 — Rogue Console Game (Java)

## Описание проекта
**AP1_Jv_P01** — консольная игра жанра Rogue/Arcade, написанная на Java с использованием библиотеки Lanterna для текстового интерфейса.
Проект выполнен в объектно-ориентированной парадигме с чётким разделением логики, интерфейса и данных. Основной фокус — архитектура, процедурная генерация уровней, интерактивный игровой цикл и расширяемость.

## Игровые возможности
- Пошаговая механика: игрок и враги ходят по очереди
- Генерация уровней: комнаты, коридоры, связность
- Алгоритмы видимости и скрытых областей
- Разнообразные враги и предметы
- Система характеристик персонажа
- Инвентарь с предметами и экипировкой
- Бои, урон, здоровье
- Лут и взаимодействие с окружением
- Сохранение/загрузка состояния в JSON
- Консольный интерфейс на Lanterna

## Используемые технологии
- Java 21
- Lanterna — рендеринг в консоли
- Gradle
- JSON (Jackson/Gson) — сохранение состояния
- Git + GitLab
- Архитектура: Presentation / Domain / Data

## Запуск
1. Клонирование
   ```bash
   git clone https://github.com/IAmAnAlligator/AP1_Jv_P01.git
   cd AP1_Jv_P01
   ```
2. Запуск
   ```bash
   ./gradlew bootRun
   ```
